# homebrew-multikey

A [Homebrew tap](https://docs.brew.sh/Taps) for installing the MultiKey CLI tool.

## About MultiKey

MultiKey is a CLI tool that manages multiple GitHub SSH identities based on folder/repo location. It allows you to automatically use different SSH keys for different repositories or directories, making it easy to work with multiple GitHub accounts or organizations.

## Installation

To install MultiKey via this tap, run:

```bash
brew install Torres-Atlantic/homebrew-multikey/multikey
```

Or if you want to add this tap to your Homebrew installation first:

```bash
brew tap Torres-Atlantic/homebrew-multikey
brew install multikey
```
